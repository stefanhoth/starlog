## [2026.05.20]
### Added
- Custom StarLog theme (indigo primary, slate palette)
- Landing page with split hero layout and "How it works" strip
- Interview mode with full-screen flashcard view and keyboard navigation
- Job Profiles with competency coverage matrix
- Story library with search and competency tag filtering
- Gemini AI extraction for voice, file, and text input
- Automatic API key validation on the onboarding screen
- GitHub Actions CI (type check, build, audit, e2e) with branch protection
- Auto-deploy to GitHub Pages on every push to main with CalVer release tags
- Renovate for automated dependency updates
